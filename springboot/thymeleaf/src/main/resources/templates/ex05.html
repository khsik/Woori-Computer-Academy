링크 표현식

- 다른 사이트로 이동

- 기본 URL 설정 방법

- URL에 변수표현식 사용시에 반드시 앞뒤에 파이프바(|)로 감싸줘야 함.

- URL에 파라미터를 설정하는 기본 방식

- URL의 {kk} 부분에 대입할 변수를 처리하는 방식

- URL 부분의 {kk} 부분에 ${data} 으로 대입. id=${'java'} 부분을 파라미터로 적용 (값을 대입하고 사용하지 않으면 파라미터로 적용) 소괄호()가 여러개 있으면 마지막 소괄호 부분만 이름=값 설정하는 속성으로 사용. 마지막 소괄호가 아니거나, 마지막 소괄호 뒤에 문자가 더 있으면 속성으로 사용 안되고 URL에 표현됨.
