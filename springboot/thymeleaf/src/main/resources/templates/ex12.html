반복문

- th:each="str : ${list}" forEach문 구조 "변수:반복리스트" => 기본구조 - th:each="str, loop : ${list}" loop는 반복문의 속성을 가진 변수 loop의 이름은 아무거나 사용해도 된다 - 반복문의 속성 loop.index : 루프의 순서 (루프의 반복 순서, 0부터 1씩 증가) loop.count : 루프의 순서 (루프의 반복 순서, 1부터 1씩 증가) loop.size : 반복 객체의 요소 개수 loop.first : 루프의 첫번째 순서인 경우 true loop.last : 루프의 마지막 순서인 경우 true loop.odd : 루프의 홀수번째 순서인 경우 true loop.even : 루프의 짝수번째 순서인 경우 true loop.current: 현재 대입된 객체
